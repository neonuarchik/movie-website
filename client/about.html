About MovieSite

MovieSite is a simple but powerful movie web app built with HTML, CSS, JavaScript, and Bootstrap.

Users can register, log in, explore movie listings, watch trailers, leave reviews, and manage a list of favorites. Data is stored using localStorage, making the site fully client-side with no backend required.

This project is ideal for learning frontend development, UI design, and practicing building interactive components using vanilla JavaScript and Bootstrap.
